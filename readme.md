## 服务端 
___
#### Socket通讯
监听服务端Socket，使用poll检测是否有新连接<bpr>
有新连接则创建客户端Socket类完成握手
___
#### 运行逻辑
服务端主循环遍历所有Socket，对其执行poll函数<bpr>
接收所有可接受数据，并添加进入任务队列<bpr>
执行命令处理函数对命令进行解析和处理<bpr>
在处理完毕后将结果发送回客户端
___
#### 客户数据存储
使用哈希表std::unordered_map对所有数据进行存储<bpr>
类中有消息队列及其账户密码<bpr>
任意客户端Socket通过验证账户密码以获取任意客户数据
___
#### 命令处理 
解析字符串后通过哈希表索引快速定位到需要处理的用户数据<bpr>
同时将处理结果写入对应Socket的缓冲区<bpr>
当检测到关机指令时，在处理完本轮任务队列后将跳出循环，结束程序
___
## 客户端
___
在主循环中不断的获取用户的命令输入，同时将命令输入发送至服务端<bpr>
并接收服务端的信息并显示
